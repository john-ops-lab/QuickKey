# Changelog

## 1.0.2

- 修复 ZIP/DMG 发布包打开呼出键设置时因找不到本地化资源包而退出的问题。
- CI 新增发布包资源路径校验，防止开发构建正常但安装包缺失运行资源。

## 1.0.1

- 修复 macOS 26 中打开或配置呼出快捷键时可能退出的问题。
- 保留快捷键冲突检测、阻止保存和键盘位置高亮。
- 同步提供 ZIP 与 DMG 安装包。

## 1.0.0

- 收录 69 个常用 macOS 快捷键。
- 支持中文搜索、分类、收藏与复制。
- 支持完整键盘位置展示和组合键高亮。
- 支持可配置的全局呼出键与冲突检测。
- 支持 Dock 和菜单栏快速访问。
